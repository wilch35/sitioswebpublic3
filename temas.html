Etiquetas basicas de HTML



  • <!DOCTYPE html>: Declara que el documento es un documento HTML y define la versión de HTML que se está utilizando.
  • <html>: Define el elemento raíz de un documento HTML.
  • <head>: Contiene información sobre el documento, como el título de la página, los estilos y los scripts.
  • <title>: Define el título de la página, que se muestra en la pestaña del navegador.
  • <body>: Contiene el contenido visible de la página, como texto, imágenes, botones, etc.
  • <h1> a <h6>: Define los encabezados de la página, donde <h1> es el encabezado más grande y <h6> es el más pequeño.
  • <p>: Define un párrafo de texto.
  • <a>: Define un enlace a otra página web o a una sección dentro de la misma página.
  • <img>: Define una imagen.
  • <ul>: Define una lista no ordenada.
  • <ol>: Define una lista ordenada.
  • <li>: Define un elemento de lista.
  • <div>: Define una sección de la página, que se puede utilizar para agrupar y organizar contenido.
  • <span>: Define una sección de texto en línea, que se puede utilizar para aplicar estilos específicos a un fragmento de texto.

Etiquetas semanticas, listas y enlaces
Etiqueta de definición
La etiqueta de definición <dt> se utiliza para crear términos de definición en una lista de definiciones <dl>. Cada término de definición <dt> debe corresponder a un elemento de definición <dd>.
Etiqueta de descripción
La etiqueta de descripción <dd> se utiliza para crear descripciones de términos de definición en una lista de definiciones <dl>. Cada elemento de descripción <dd> debe corresponder a un término de definición <dt>.
Etiqueta de listas
La etiqueta de listas <ul> se utiliza para crear listas de elementos no ordenados. Dentro de una lista de elementos no ordenados se pueden utilizar las etiquetas <li> para especificar cada elemento de la lista. Los elementos de la lista siempre se muestran con marcadores.
Etiquetas de enlace
Las etiquetas de enlace <a> se utilizan para crear enlaces a otras páginas o a otras partes de la misma página. El atributo href de la etiqueta indica la URL del enlace. El texto entre las etiquetas de enlace se muestra como el enlace.

Flex Box CSS Grid

Flexbox y CSS Grid son dos técnicas de diseño de diseño web que permiten la creación de diseños flexibles y adaptables a diferentes tamaños de pantalla y dispositivos.

Flexbox, o "Flexible Box", es una técnica de diseño que se utiliza para la creación de diseños flexibles y adaptables en una sola dimensión (ya sea horizontal o vertical). Se utiliza principalmente para el diseño de elementos individuales, como por ejemplo la distribución de elementos en una fila o columna. Flexbox se basa en un modelo de cajas flexibles y proporciona un conjunto de propiedades CSS para controlar la distribución, el tamaño y el orden de los elementos dentro del contenedor.

Por otro lado, CSS Grid es una técnica de diseño que permite la creación de diseños de cuadrícula bidimensionales (en dos dimensiones) que se adaptan automáticamente a diferentes tamaños de pantalla. Con CSS Grid se pueden crear diseños complejos y dinámicos que responden a los cambios en el tamaño de la pantalla y la disposición de los elementos. CSS Grid se basa en un modelo de cuadrícula y proporciona un conjunto de propiedades CSS para definir la estructura de la cuadrícula, así como para controlar la distribución y el tamaño de los elementos dentro de ella.

En resumen, Flexbox es ideal para diseños de una dimensión, mientras que CSS Grid es más adecuado para diseños de dos dimensiones y diseños más complejos. Ambas técnicas son muy útiles y complementarias para el diseño web moderno y adaptable.
